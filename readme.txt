Dataset contains 74 sample raw readings with Flir Lepton 3 thermal module

raw_im_x.jpeg (where x is number between 1 and 74) contains raw reading normalized and adjusted for preview
raw_im_x.tiff (where x is number between 1 and 74) contains raw reading in 16 bit tiff format
temp_reading_x.txt (where x is number between 1 and 74) contains high precision temperature reading in txt format.
